# Changelog

## 3

### 3.5
_2020-08-20_  
[Direflow v3.5 Release Notes](https://medium.com/@simonlhoiberg/direflow-v3-4-0-release-notes-60e1caff3805)

#### Added
- Support for `direflow-config.json`. See the documentation [here](/configuration)
- Ability to include additional static files in build [#157](https://github.com/Silind-Software/direflow/issues/157)
- Support for serialized data via attributes [#127](https://github.com/Silind-Software/direflow/issues/127)
- Parsing boolean values from attributes [#125](https://github.com/Silind-Software/direflow/issues/125)

#### Changed
- A new `DireflowPromiseAlike` is returned from `DireflowComponent.create`.

#### Deprecated
*These are now supported as part of `direflow-config.json`*
- Passing `{ filename, chunkFilename, react, reactDOM }` to webpack config function
- Using `--split` and `--vendor` as CLI arguments
- polyfill-loader plugin

### 3.4
_2020-03-13_  
[Direflow v3.4 Release Notes](https://medium.com/@simonlhoiberg/direflow-v3-4-0-release-notes-60e1caff3805)

#### Added
- Split Direflow Components into multiple bundles [#53](https://github.com/Silind-Software/direflow/issues/53)
- Configure Direflow Components using `defaultProps` and `@DireflowConfiguration` decorator [#57](https://github.com/Silind-Software/direflow/issues/57)
- Create NPM-ready boilerplate [#56](https://github.com/Silind-Software/direflow/issues/56)
- Option to use `async` when using `external-loader` plugin [#58](https://github.com/Silind-Software/direflow/issues/58)
- Build a regular React component library from Direflow Setup.
- Ability to provide custom CDN for `React` and `ReactDOM` modules, thus ability to disable lazyloading.
- Added `useExternalSource` hook [#137](https://github.com/Silind-Software/direflow/issues/137)
- Added ability to choose where to inject styles
- Optionally disable scoped styled [#52](https://github.com/Silind-Software/direflow/issues/52)

#### Changed
- New structure for configuring a Direflow Component
- Added new package: `direflow-scripts`. Moved everything webpack and build-related to this package.

### Fixed
- Issues with Material UI [#106](https://github.com/Silind-Software/direflow/issues/106) and [#105](https://github.com/Silind-Software/direflow/issues/105)
- Styled Components version 5> not rendering [#94](https://github.com/Silind-Software/direflow/issues/94)
- Path not resolving properly on Windows OS [#92](https://github.com/Silind-Software/direflow/issues/92), [#138](https://github.com/Silind-Software/direflow/issues/138) and [#134](https://github.com/Silind-Software/direflow/issues/134)

## 2

### 2.2
_2020-01-06_

#### Added
- New custom implementation of handling event retargeting  
  Fixes a range of common bugs with event propagation and support for IE
- Properties take precedence over attributes if both are set [#31](https://github.com/Silind-Software/direflow/pull/31)

#### Fixed
- Import modules from without the `./src` folder [#24](https://github.com/Silind-Software/direflow/issues/24)
- Issue with reattaching Shadow Root [#27](https://github.com/Silind-Software/direflow/issues/27)
- Multiple components added to the same shadow DOM [#28](https://github.com/Silind-Software/direflow/issues/28)

### 2.1
_2019-12-22_

#### Changed
- Both attributes and properties are now derived from the same object  
  The separation is made implicit - the `attribute` field and `setAttribute()` method  
  can now be omitted on the Direflow configuration

#### Changed

#### Deprecated

#### Fixed
- Multiple Web Components referenes the same property object ([#12](https://github.com/Silind-Software/direflow/issues/12))
