## Create new project
Get started by running the command
```console
npx create-react-web-component
```
Or install the package globally with yarn or NPM
```console
yarn global add create-react-web-component
```
```console
npm i -g create-react-web-component
```

This will bootstrap a new project for you - you can choose whether you want to use **TypeScript** or regular **JavaScript**.  
Now use the following commands:
```console
cd <project-folder>
yarn install
yarn start
```
Your project will start running on `localhost:3000` and your browser opens a new window
