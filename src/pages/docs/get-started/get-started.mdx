## Install CLI
Start by installing direflow-CLI
```console
npm i -g direflow-cli
```

## Create new Direflow Component
Now, use this command to create a new Direflow Component
```console
direflow create -c
```
Now you can choose a name for your component and a prefered language. Currently both JavaScript and TypeScript is supported.  
A folder will be created at your current location.  
  
Now, cd into your new project and use the following commands
```console
cd <project-folder>
yarn install
yarn start
```
Your Direflow Component will start running on `localhost:3000` and your browser opens a new window

## Create new Direflow Project
In order to create a new Direflow Project, use this command
```console
direflow create -p
```
Now you can choose a name for your project and a prefered language. Currently both JavaScript and TypeScript is supported.  
Note, that you can still choose a different language on individual Direflow Components.

A folder will be created at your current location.  
  
Now you can create your first component by cd'ing into this folder and use
```console
direflow create
```
The Direflow Component will be created in `direflow-components/your-component`
